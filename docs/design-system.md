# Design System Documentation

## Cores da Marca
Nossa paleta foi escolhida para evocar energia, confiança e calor local.

### Brand
- **Primary (Laranja Sinal):** `#FF6B35` - Energia, Ação, Calor.
- **Secondary (Azul Profundo):** `#004E89` - Confiança, Profissionalismo, Estabilidade.
- **Accent (Amarelo Dourado):** `#F7B801` - Otimismo, Destaque.

### Neutros
- **White:** `#FFFFFF`
- **Black:** `#1A1A1A` (Nunca usar #000000 puro)
- **Gray Scale:** Cool gray base (Tailwind Gray)

## Tipografia
Usamos uma escala tipográfica agressiva para o estilo Brutalismo de Energia Local.

- **Massive Heading:** 12rem (192px) - Usado apenas na Hero Section
- **Display:** Fontes bold para títulos
- **Body:** Inter para legibilidade

## Formas e Bordas
- **Radius Sharp (2px):** Padrão para cards e botões para dar um ar técnico e preciso.
- **Círculos:** Apenas para avatares e badges de status.

## Animações
O design deve parecer vivo.

- **Spring Physics:** Todas as transições devem usar física de mola (stiffness: 100-400).
- **Micro-interações:** Feedback imediato em todos os cliques/hovers.
- **Scroll Triggers:** Elementos entram na tela suavemente conforme o scroll.
